### Objective

This project aims to:

- Build a **high-impact personal portfolio** using modern frontend and 3D technologies  
- Combine **visual storytelling** with **professional developer branding**
- Demonstrate hands-on expertise in **React, 3D rendering, animation, and UI engineering**
- Provide a visually engaging, recruiter-friendly alternative to traditional portfolios
- Serve as a scalable foundation for showcasing projects and case studies

---

### Key Features

| Area | Implementation |
|------|----------------|
| 3D Rendering | Interactive experiences built using React Three Fiber with Drei abstractions |
| Animations | Scroll-based transitions and micro-interactions implemented with Framer Motion |
| UI & Styling | Responsive and accessible interface designed with TailwindCSS |
| Contact Handling | Functional contact form integrated via EmailJS |
| UI Enhancements | Reusable and polished components using Aceternity UI and Magic UI |
| Tooling & Builds | Fast local development and optimized production builds powered by Vite |


---

### Tech Stack

| Tool / Library        | Purpose |
|----------------------|---------|
| React                | Core frontend framework |
| Vite                 | Fast build tool and dev server |
| TailwindCSS          | Utility-first styling |
| React Three Fiber    | Declarative 3D rendering with Three.js |
| Drei                 | Helpers and abstractions for R3F |
| Framer Motion        | Animations and transitions |
| EmailJS              | Contact form email integration |
| Aceternity UI        | Advanced UI components |
| Magic UI             | Prebuilt interactive UI elements |
